## 📊 Project Overview

This project showcases a **Pizza Sales Report Dashboard** created using Power BI. It provides detailed insights into sales performance, customer ordering behavior, and product popularity across different categories and sizes.

The dashboard is designed to help businesses analyze trends, optimize inventory, and improve sales strategies.

---

## 🚀 Key Metrics

* **Total Sales:** 817.86K
* **Total Pizzas Sold:** 50K
* **Total Orders:** 21K
* **Average Orders per Customer:** 2
* **Average Order Value:** 38.31

---

## 📌 Dashboard Insights

### 🔹 Sales Trends

* **Day-wise Sales:** Tracks daily fluctuations in sales performance
* **Monthly Sales:** Identifies peak and low-performing months

### 🔹 Product Analysis

* **Top 5 Best-Selling Pizzas:**

  * The Thai Chicken Pizza
  * The Barbecue Chicken Pizza
  * The California Chicken Pizza
  * The Classic Deluxe Pizza
  * The Spicy Italian Pizza

### 🔹 Category Performance

* Sales distribution across pizza categories:

### 🔹 Size-wise Sales

  * M
