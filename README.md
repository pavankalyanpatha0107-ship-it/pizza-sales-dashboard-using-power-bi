# 🍕 Pizza Sales Dashboard (Power BI)

## 📊 Overview

This project is an interactive **Pizza Sales Dashboard** built using **Power BI**. It provides insights into sales performance, order trends, and customer preferences across different pizza categories, sizes, and time periods.

---

## 🚀 Features

* **Total Sales Analysis**
  Displays overall revenue generated.

* **Order & Quantity Insights**

  * Total number of orders
  * Total pizzas sold
  * Average orders per customer

* **Average Order Value (AOV)**
  Helps understand customer spending behavior.

* **Time-Based Analysis**

  * 📅 Day-wise sales trends
  * 📆 Monthly performance comparison

* **Category-wise Sales**
  Breakdown of sales by pizza categories:

  * Classic
  * Supreme
  * Chicken
  * Veggie

* **Top 5 Best-Selling Pizzas**
  Identifies the most popular pizzas.

* **Sales by Pizza Size**
  Distribution of sales across sizes (S, M, L, XL, XXL).

---

The dataset and preview are present in this repository

---

## 🛠️ Tools & Technologies

* **Power BI** – Data visualization
* **Excel / CSV Dataset** – Data source
* **DAX (Data Analysis Expressions)** – Calculations and measures

---

## 📈 Key Insights

* Large size pizzas contribute the highest share of revenue.
* Certain months show peak sales trends, indicating seasonal demand.
* A few top-selling pizzas generate a significant portion of total revenue.
* Daily sales fluctuate, showing potential for demand forecasting.
